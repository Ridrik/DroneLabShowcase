About

DroneLab is a C++ based simulator and visualizer tool for drone guidance, navigation, and control (GNC). It supports live and replay sessions with full telemetry history, plots, and visualizations.

  • GNC core: rigid-body dynamics with RK4 integration, discrete EKF for state estimation, and cascade PID controllers
  • User friendly configurable editor for controllers, drone parameters, sensors, estimators, rules, environments, waypoints and more
  • Waypoint following and return-to-launch modes, with more flight modes planned
  • Simulation control: pause, slow or speed up even on a live session
  • Interactive features: mouse hover and add waypoints dynamically, or toggle sensors On/Off
  • Visual outputs: plots, telemetry panels, minimap, 3D trajectory trails, ghost drone estimates, covariance ellipsoids, and more
  • Automatic saving and replay of runs in near-original detail
